## Prerequisites

- A modern browser (Chrome, Firefox, Edge)
- Node.js 18+ for running scripts outside the browser
- A code editor (VS Code recommended)

---

## 🚀 Quick Start

```html
<!-- browser: inline script -->
<script>
  console.log("Hello, JavaScript!");
</script>
```

```js
// Node.js: run with  node hello.js
console.log("Hello, JavaScript!");
```

```bash
node --version   # v18+
node hello.js
```

## 🔑 Why JavaScript?

| Feature | Description |
|---------|-------------|
| **Ubiquitous** | Runs in every browser + server (Node) + mobile (RN) + desktop (Electron) |
| **Dynamic** | Flexible typing, first-class functions, prototypal inheritance |
| **Event-driven** | Non-blocking I/O model perfect for UI and network work |
| **Huge Ecosystem** | 2 million+ npm packages |
| **Active Standard** | New features every year (ES2015 → ES2024+) |

## 💡 Learning Tips

- **Code along** — open DevTools console and experiment with every snippet.
- **Build things** — each concept sticks better with a small project.
- **Read the spec** — MDN Web Docs is the authoritative reference.
- Follow the order; later chapters build on earlier ones.
